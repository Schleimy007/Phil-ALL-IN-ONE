Datenschutz

1. Datenschutz auf einen Blick
Wir nehmen den Schutz Ihrer persönlichen Daten sehr ernst. Diese Website speichert Daten (wie Notizen) fast ausschließlich lokal in Ihrem Browser (localStorage).

2. APIs von Drittanbietern
Für das Wetter-Tool nutzen wir die Open-Meteo API. Hierbei wird Ihre IP-Adresse an deren Server übermittelt, um die Daten abzurufen. Open-Meteo speichert nach eigenen Angaben keine persönlichen Daten.

3. Kontaktformular
Das Kontaktformular nutzt eine "mailto:" Verknüpfung. Es werden keine Daten auf unseren Servern gespeichert; die Daten werden direkt an Ihr lokales E-Mail-Programm übergeben.
